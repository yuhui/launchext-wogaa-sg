No configuration necessary

WOGAA collects data across Singapore government agency websites to improve government services with data.

Learn more about WOGAA.

How this Extension works

  1. Create a new Rule or open an existing Rule, for example, one with a "Library Loaded" Event.
  2. Add the "Implement Informational Service" Action.
  3. (optional) If you want to track transactions, add the "Implement Transactional Service" Action to your appropriate Rule(s) and configure it as instructed.
  4. (optional) If you want to correlate Sentiments with your agency's data, add the "Correlate Sentiments with Agency's Data" Action to your appropriate Rule(s) and configure it as instructed.
Important! Works with Singapore government-registered agency websites only.

WOGAA can only be used in websites that have been registered with WOGAA. Implementing WOGAA in unregistered websites has no effect at all.
